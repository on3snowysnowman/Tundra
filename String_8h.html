Detailed Description

Functions and data structures for string manipulation.

Author
Joel Height (On3Sn.nosp@m.owyS.nosp@m.nowma.nosp@m.n@gm.nosp@m.ail.c.nosp@m.om)

This header defines a set of utilities for creating, managing, and manipulating strings. It includes functions for string allocation, deallocation, concatenation, comparison, and other common string operations.

Adds a sequence of chars to the string.

The number of characters to add must be specified.

It is irrelevant if chars is null terminated or not, just ensure that if it is, the terminator character is NOT included in final count of num_chars.

Parameters
stringString to modify.

IF init_char_capacity is 0, no memory will be allocated until the initial character(s) are added.
