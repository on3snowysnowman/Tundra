Detailed Description

Internal-only interface for low-level key state management.

This internal struct provides direct control over the raw and processed key state flags in Tundra::InputManager. It is intended for use by the core engine systems that process platform input events and should not be accessed by end-users.

static void Tundra::Internal::InputManager::reset_raw_pressed_keys ( )
inlinestatic

Resets all raw key press flags.

Called at the end of each input processing cycle to clear the temporary state that tracks which keys are pressed this frame. This is used to distinguish key press events from held states.
