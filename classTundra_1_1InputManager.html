Detailed Description

Automatically tracks inputs from the keyboard, as well as processing user specified callback invocation from said pressed keys.

The InputManager class manages the registration and invocation of callbacks tied to key presses. It supports storing member function callbacks associated with individual keys, removing them by ID, and invoking them at runtime.
